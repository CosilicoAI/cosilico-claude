# cosilico-claude

Claude Code plugin for Cosilico's synthetic microdata and microsimulation tools.

## Skills

- **microplex** - Synthetic microdata quality evaluation, synthesizer training, authenticity vs privacy metrics
- **design-system** - Cosilico design system patterns

## Note

Encoding infrastructure (RAC, atlas, autorac) has moved to the [rules-foundation](https://github.com/RulesFoundation) plugin.
